## 接线图

```
                    你的电源 PCB (45×38mm)
                    ╔════════════════════╗
    LRS 5V红 ───→  ║ 5V_IN    [焊接]    ║
    LRS GND黑 ──→  ║ GND      [焊接]    ║
                    ║                    ║
                    ║  [LC滤波]          ║
                    ║  10μH→470μF       ║
                    ║                    ║
                    ║  [AMS1117模块]     ║
                    ║  插在排母上        ║
                    ║                    ║
                    ║ 5V_SERVO  [焊接]──→║──→ PCA9685 #1 V+ (+GND)
                    ║ GND       [焊接]──→║──→ PCA9685 #1 GND
                    ║                    ║
                    ║ 5V_LOGIC  [焊接]──→║──→ PCA9685 #1/#2 VCC (逻辑)
                    ║ GND       [焊接]──→║      + ASRPRO 5V
                    ║                    ║
                    ║ 3.3V      [焊接]──→║──→ STM32 3.3V
                    ║ GND       [焊接]──→║──→ STM32 GND
                    ╚════════════════════╝
```

### 详细连接

| 电源板焊盘 | 线径 | 焊到 | 备注 |
|-----------|------|------|------|
| J1 5V_IN + GND | ≥1.0mm² | LRS 5V 输出端子 | 电源总输入 |
| J2 5V_SERVO + GND | ≥1.0mm² | PCA9685 V+ 端子 + GND | 舵机电源（两块PCA9685并在一起焊） |
| J4 5V_LOGIC + GND | 0.5mm² | PCA9685 VCC脚 + ASRPRO VCC | 电感前取电，不受舵机干扰 |
| J3 3.3V + GND | 0.3mm² | STM32 3.3V脚 + GND | 单片机供电 |

### 星型接地关键规则

所有 GND 焊盘在 PCB 内已做底层铜箔汇流，但外部接线也要遵守：
- **正确的**：每个模块的 GND 分别飞线到电源板 GND 焊盘
- **错误的**：PCA9685 GND → STM32 GND → 电源板（链式接地会引入共阻抗噪声）

## I2C 上拉电阻（在主板补焊）

PCA9685 模块没有上拉电阻。在现有主板上飞两个 4.7KΩ (1/4W)：

```
SDA排针 ──[4.7KΩ]── 3.3V排针
SCL排针 ──[4.7KΩ]── 3.3V排针
```

用 3.3V 不是 5V，因为 STM32 I/O 是 3.3V 电平。

## 导入立创EDA

1. 打开立创EDA专业版
2. 文件 → 导入 → 嘉立创EDA(标准版)
3. 选择 `机械时钟电源板_完整工程.zip`
4. 检查元件封装映射是否正确
5. 如有文本偏移手动调一下
6. 导出 Gerber → JLC 下单

## 备注

- ASRPRO 吃 5V 不是 3.3V，接 J4 (5V_LOGIC)
- DS1302 自带纽扣电池，不掉时间
- ST-Link 烧录器 3.3V 断开不用，全由 LRS 统一供
